Title: ClinicBook - Multimodal AI Healthcare Platform
Tags: Flask, Python, Google Gemini, Bootstrap, Jinja2, Web Speech API, SQLite
Content:
<p>A comprehensive web-based platform that transforms appointment booking into an intelligent, conversational experience. <strong>ClinicBook</strong> features a <strong>Context-Aware Agentic AI</strong> capable of reading, writing, listening (Voice), and reasoning to manage healthcare needs seamlessly.</p><p>Find the repository on <a href="https://github.com/Mfaj-cod/ClinicBook.git">GitHub</a>.</p><h5>Key Features</h5><h5>Agentic AI Health Assistant (Powered by Gemini)</h5><p><em>A Hybrid AI system combining Agentic Tool Use with Retrieval-Augmented Generation (RAG).</em></p><ul><li><strong>Voice Interface</strong>: Built-in <strong>Speech-to-Text</strong> allows doctors and patients to speak commands naturally (e.g., "Book an appointment" or "Generate 5 slots").</li><li><strong>Hybrid RAG System</strong>: <ul><li><strong>Dynamic Data</strong>: Queries the SQLite database for live schedules and doctor availability.</li><li><strong>Static Knowledge</strong>: Instantly answers policy questions (Refunds, Insurance, Hours) from a local Knowledge Base without unnecessary database calls.</li></ul></li><li><strong>Context-Aware Identity</strong>: Automatically detects User Role (Doctor/Patient) and Current Date to handle relative queries like "Show me tomorrow's schedule".</li></ul><h5>Doctor Portal</h5><ul><li><strong>Voice-Powered Management</strong>: Dictate commands like "Create slots for next Monday at 10 AM" to bulk-generate schedule capacity.</li><li><strong>Smart Dashboard</strong>: Ask "Who is visiting me today?" to get a filtered, real-time patient list.</li><li><strong>Workflow Automation</strong>: Mark appointments as "Completed" directly through chat.</li><li><strong>Secure Access</strong>: Session-based isolation ensures doctors only manage their own data.</li></ul><h5>Patient Portal</h5><ul><li><strong>Symptom-to-Specialist</strong>: AI intelligently maps symptoms (e.g., "chest pain") to the correct specialist (e.g., <em>Cardiologist</em>).</li><li><strong>One-Click Booking</strong>: Streamlined booking flow with email confirmation.</li><li><strong>Transparency</strong>: Clear visibility of appointment status, fees, and clinic policies via the AI assistant.</li></ul><h5>Technology Stack</h5><ul><li><strong>Backend</strong>: Flask (Python 3.x)</li><li><strong>AI Engine</strong>: Google Gemini 2.5 Flash</li><li><strong>AI Architecture</strong>: Agentic Tools (Function Calling for DB Writes/Reads), Lightweight RAG (Context injection for policy documents), System Prompting (Role-based behavior control).</li><li><strong>Frontend</strong>: Bootstrap 5 & Jinja2, Web Speech API (Native JavaScript)</li><li><strong>Database</strong>: SQLite (Transactional)</li></ul><h5>AI Capabilities Breakdown</h5><p>The system uses a <strong>Router Architecture</strong> to decide how to handle a user query:</p><ul><li><strong>Knowledge Query</strong> (e.g., "Do you accept insurance?")<ul><li><strong>Source:</strong> <code>clinic_policies.txt</code></li><li><strong>Action:</strong> RAG Response (No DB Call)</li></ul></li><li><strong>Data Query</strong> (e.g., "Find a Urologist")<ul><li><strong>Source:</strong> <code>search_doctor_by_specialization</code> Tool</li><li><strong>Action:</strong> DB Read → Returns List with [Hidden IDs]</li></ul></li><li><strong>Action Command</strong> (e.g., "Book the 10 AM slot")<ul><li><strong>Source:</strong> <code>book_appointment_by_patient</code> Tool</li><li><strong>Action:</strong> DB Write → Confirmation</li></ul></li></ul>

Title: Agentic CMS Portfolio
Tags: Python, Flask, LangChain, Google Gemini, Bootstrap, SQLite, HTML5, Jinja2, Pydantic, Custom CSS
Content:
<p>A self-updating portfolio website powered by LangChain and Google Gemini.</p><h5>Overview</h5><p>This is not a static portfolio. It is an <strong>Agentic Content Management System</strong>. Instead of manually writing HTML for every new project or blog post, a LangChain-based AI Agent is utilized. Through a secure Admin Dashboard, natural language prompts are provided, and the Agent:</p><ul><li><strong>Analyzes</strong> the intent (Blog vs. Project vs. Bio).</li><li><strong>Generates</strong> structured content (HTML, Tags, Links).</li><li><strong>Formats</strong> it for the frontend.</li><li><strong>Saves</strong> it to the SQLite database upon approval.</li></ul><h5>Key Features</h5><ul><li><strong>Intelligent Orchestrator:</strong> Uses LangChain and Pydantic to enforce strict JSON schemas, ensuring the AI never breaks the UI layout.</li><li><strong>Monochrome Premium UI:</strong> A sleek, high-contrast dark theme built with Bootstrap 5 and custom CSS gradients.</li><li><strong>Secure Admin Panel:</strong> Protected route with session-based authentication to manage content.</li><li><strong>Dynamic Content:</strong><ul><li><strong>Projects:</strong> Auto-formatted cards with "Read More" modals and GitHub links.</li><li><strong>Blogs:</strong> Technical articles structured with HTML headers and lists.</li><li><strong>About Me:</strong> AI-generated bio updates based on skills and experience.</li></ul></li><li><strong>History Logging:</strong> Automatically logs all generated content to a local history.txt file (UTF-8 supported) for backup.</li></ul><h5>Tech Stack</h5><ul><li><strong>Backend:</strong> Flask (Python)</li><li><strong>Database:</strong> SQLite (Native JSON storage)</li><li><strong>AI Logic:</strong> LangChain, Google Generative AI (Gemini 1.5 Flash)</li><li><strong>Frontend:</strong> HTML5, Jinja2, Bootstrap 5, Custom CSS</li><li><strong>Validation:</strong> Pydantic (for structured AI output)</li></ul><h5>How to Use the Agent</h5><p>Users can navigate to /admin and log in.</p><p>To Add a Project:</p><p><em>Example Prompt:</em> "It's a project called 'ClinicBook'. It uses Flask and Gemini to help patients book appointments."</p><p><em>Result:</em> The Agent categorizes it as a Project, extracts the tech stack, formats the description into HTML, and adds the GitHub link.</p><p>To Write a Blog:</p><p><em>Example Prompt:</em> "Write a blog post about the importance of Vector Databases in RAG applications."</p><p><em>Result:</em> The Agent writes a structured article with headings and bullet points.</p><p>To Update Bio:</p><p><em>Example Prompt:</em> "Update my about section. I am now focusing on Agentic Workflows and Generative AI."</p><p><em>Result:</em> The Agent rewrites the bio and updates the skills tags.</p>
